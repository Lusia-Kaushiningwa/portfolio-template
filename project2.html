The Smart Quiz App is an interactive web application that allows users to answer multiple-choice questions, track their score, and receive instant feedback.The goal. of this project was to build a fun, engaging, and fully functional quiz system using pure front-end technologies.

This project focuses on logic building, DOM manipulation, and user interaction. It simulates a real quiz environment with timed questions, score tracking, and dynamic question rendering.

The app was built using HTML, CSS, and JavaScript without external frameworks, which helped strengthen my understanding of core web development concepts.
